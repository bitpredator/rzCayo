# rzCayo
For load Cayo Perico in your server Fivem, start these folders in your server.cfg and set this to your server.cfg : sv_enforceGameBuild 2189 !

So in your server.cfg :


#### #### Cayo Perico #### ####

sv_enforceGameBuild 2189

ensure rzCayo

ensure rzCayoBridge
